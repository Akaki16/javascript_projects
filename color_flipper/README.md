# Color Flipper

## Description

In this project user is able to pick up several random colors by clicking on "Pick a Color" button

### JavaScript Features Used includes:

1) `arrays`

2) `document.getElementById()`

3) `document.querySelector()`

4) `addEventListener()`

5) `Math.floor()`

6) `Math.random()`

7) `array.length`

8) `functions`
